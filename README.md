# Tire Size API — Help & Integration Docs

Integration guides and ready-to-use examples for the Tire Size API
(`https://tire.vdim.app/api/v1`). Get a free API key at
[account-tire.vdim.app](https://account-tire.vdim.app/signin/signup).

## Integrations

- **[n8n](integrations/n8n/)** — three import-ready n8n workflows (cascading
  vehicle lookup, tire dimensions lookup, reverse lookup) plus a step-by-step
  setup guide.

Need help? Contact support via <https://tire.vdim.app/contact.html>.
